# Prompt Examples

下面这些 prompt 可以直接复制到 Codex 里使用。

## 只做配图规划

```text
Use $jacky-growth-illustrations 先不要生图。
请分析下面这篇文章哪里值得配图，输出 5 张左右的 shot list。
每张图写清楚：
- 放在哪个段落后
- 图的主题
- 核心意思
- 结构类型
- Jacky 小芽哥在图里做什么
- 建议元素
- 建议中文标注词

<粘贴文章>
```

## 文章正文配图

```text
Use $jacky-growth-illustrations 把下面这篇文章生成 4 张 Jacky 小芽哥正文配图。
要求：16:9 横版、纯白背景、黑色手绘线稿、少量红橙蓝中文手写批注。
角色必须是白色豆豆/小黑体型，头顶嫩芽清楚，细手细腿，有知识工位气质。
可以有小墨镜，但不要太大、太凶或遮住嫩芽。

<粘贴文章>
```

## 长文配图策略

```text
Use $jacky-growth-illustrations 给这篇长文做配图策略。
不要平均配图，只挑认知锚点：核心判断、输入输出闭环、前后对比、常见坑、承接路径。
默认 6-8 张，先输出 shot list，不要生成图片。

<粘贴文章>
```

## 单个观点生成一张图

```text
Use $jacky-growth-illustrations 为这个观点生成一张 16:9 正文配图：

真正的成长不是收藏更多方法，而是让一个动作反复发芽。

画面要怪诞但清爽，Jacky 小芽哥必须承担核心动作。
可以戴小墨镜，有一点潇洒哥气质。
中文标注最多 5 个，短一点。
```

## 工作流主题

```text
Use $jacky-growth-illustrations 为“把一条原始素材加工成流量、信任、转化三种内容”生成一张图。
不要画正式流程图，不要复刻旧案例。
请重新发明一个新的低科技隐喻，让 Jacky 小芽哥参与核心动作。
```

## 改图：增强潇洒哥气质

```text
Use $jacky-growth-illustrations 这张图方向对，但角色不够有活力。
请保持核心意思不变，重生成一版：
- 白色豆豆/小黑体型
- 头顶嫩芽清楚
- 墨镜可选但要小
- 姿态更松弛自信
- 不要树桩、普通人形或商业贴纸感
```

## 改图：去掉标题

```text
Use $jacky-growth-illustrations 帮我编辑这张图。
去掉左上角的“Workflow / 流程图”标题和下划线，其他内容保持不变。
不要新增任何文字或物件。
```

## 生成一组风格样片

```text
Use $jacky-growth-illustrations 输出 5 个不同主题的 Jacky 小芽哥正文配图效果。
主题分别覆盖：信息过载、产品验证、内容复利、一人公司、信任建立。
每张单独生成，不要拼成一张。
```
