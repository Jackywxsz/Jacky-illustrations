# Jacky 小芽哥 IP

## 角色定义

Jacky 小芽哥是“Jacky无限生长”的固定视觉 IP。

默认每张图都要出现 Jacky 小芽哥。它是一个白色豆豆体型、头顶嫩芽、细手细腿的知识工位角色，可以戴同款小黑墨镜呈现“潇洒哥”气质。它不是吉祥物、贴纸或儿童卡通，而是正在认真参与内容系统、知识沉淀和行动闭环的荒诞工作者。

## 外形

- 白色圆润豆豆/小黑体型，身体是一整块柔软的圆角形。
- 黑色手绘线稿，不使用大面积彩色填充。
- 头顶固定长出两片嫩芽，这是最重要识别点。
- 脸很简单：小圆点眼睛、小嘴，或戴小黑墨镜。
- 墨镜是同一副小黑墨镜：黑色镜片、粗黑框、白色斜高光，可正戴或推到头上。
- 墨镜只作为人格配件，尺寸要小，不能遮住嫩芽，也不要换成圆眼镜、飞行员眼镜或其他款式。
- 细手、细腿，脚小而稳。
- 每个动作默认要有清楚的手臂和手；手臂从身体侧面或下侧长出，不从头顶、嫩芽或脸上长出。
- 动作简洁但有活力：站得松弛、指向上方、跑、跳、抱笔记本、敲键盘、浇水。
- 轮廓略微不规则，有随手画的笔触感。
- 可以用极少量绿色点亮嫩芽；如果用户要求纯黑线，则嫩芽也只用黑线。

## 灵感符号

- Jacky 小芽哥的灵感符号不是彩色圆点。
- 正确画法是“灵感三线”：3 条黑色短线，像突然有想法。
- 可以画在头部旁边、举起的手指旁边、便签旁边或新芽旁边。
- 禁止画成橙/蓝/白三个圆点，禁止画成 UI 状态点、批注框、叉号或装饰气泡。

## 性格

- 很认真，但做的事有点荒诞。
- 活力满满，有“潇洒哥”的松弛自信。
- 像一个低调但很会干活的内容系统操作员。
- 冷幽默，不卖萌。
- 有点笨拙，但不蠢。
- 代表“持续生长”：从信息、行动、复盘和积累里长出新东西。

## 常见职责

让 Jacky 小芽哥承担核心动作：

- 抱着笔记本记录。
- 戴墨镜指向一个关键判断。
- 在知识工位前输入、整理、输出。
- 给“想法种子”浇水。
- 把零散信息种进土里。
- 修剪多余枝杈。
- 把旧内容搬进养分箱。
- 在内容机器里推动“生长”杆。
- 顶着嫩芽穿过系统断点。
- 把行动清单插成一排苗。
- 给失败样本贴标签堆肥。
- 拉线汇聚信息源。
- 卡在断点里。
- 牵着承接路径。
- 搬砖、搭桥、开门、分拣、记录。

## 禁止

- 不要画成树桩、普通人形男孩、黑豆、影子怪物、机器人、动物或抽象种子。
- 不要丢掉白色豆豆体型、细手细腿和头顶嫩芽。
- 不要把嫩芽画得太小或被墨镜遮住。
- 不要把墨镜画成凶狠、黑社会、油腻、过度装饰或不同款式。
- 不要把灵感符号画成彩色圆点三连；只能使用黑色短线式灵感三线。
- 不要让动作缺手、缺手臂，或让手从头顶/嫩芽附近长出。
- 不要回到彩色贴纸感：厚描边、强腮红、大面积萌系表情都不要。
- 不要画成儿童卡通角色。
- 不要给 Jacky 小芽哥复杂服装、头发、帽子、鞋服细节。
- 不要让 Jacky 小芽哥只是站在角落里看。
- 不要让 Jacky 小芽哥抢走结构表达。

## 判断标准

如果去掉 Jacky 小芽哥，图的核心隐喻还能完全成立，说明角色太装饰了；要重写提示词，让 Jacky 小芽哥成为动作主体。

如果遮住头顶嫩芽后完全没有“Jacky无限生长”的识别度，说明 IP 识别点不够。

如果第一眼像树桩、普通人、宠物或商业贴纸，说明形态失败。
