# Jacky 小芽哥 reference v4

This folder contains the current visual reference set for the Jacky 小芽哥 IP.

Use these images when generation drifts on:

- white rounded bean body consistency
- same black sunglasses style
- top green two-leaf sprout
- black short-line inspiration mark
- visible arms and hands in action poses

Files:

- `01-identity-master-v4.png`: main identity board
- `02-turnaround-v4.png`: front/side/back and visual anchors
- `03-accessory-extension-v4.png`: expression, action, and accessory extension sheet
- `04-article-baseline-v4.png`: article illustration baseline

Rules:

- Inspiration marks are three short black lines only.
- Do not use orange/blue/white colored dot trios as inspiration marks.
- Every action pose should show recognizable arms and hands.
- Sunglasses can be worn on the face or pushed onto the head, but must remain the same black sunglasses style.
